# Les fichiers d'Olivier

## Description

Quelques fichiers de configuration possibles pour des outils quotidiens tels que : 

  * Tmux
  * Vim
  * Xorg
  * Zsh
  * etc.

D'autres viendront au fur et à mesure.

## Dépendances

Sont nécessaires : 

  * zsh
  * vim
  * htop
  * most
  * tmux
  * elinks
  * openbox
  * redshift
  * xscreensaver
